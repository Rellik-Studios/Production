\doxysubsection{Detailed Description}
All story state information is included in the \mbox{\hyperlink{class_ink_1_1_runtime_1_1_story_state}{Story\+State}} class, including global variables, read counts, the pointer to the current point in the story, the call stack (for tunnels, functions, etc), and a few other smaller bits and pieces. You can save the current state using the json serialisation functions To\+Json and Load\+Json.
