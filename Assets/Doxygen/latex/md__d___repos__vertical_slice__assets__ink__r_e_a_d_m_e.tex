\item {\bfseries{ink player}}\+: Provides a powerful \href{https://github.com/inkle/ink-unity-integration/blob/master/Documentation/InkPlayerWindow.md}{\texttt{ Ink Player Window}} for playing and debugging stories.
\item {\bfseries{Auto compilation}}\+: Instantly creates and updates a JSON story file when a {\ttfamily .ink} is updated.

\hypertarget{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md13}{}\doxysubsection{Compilation}\label{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md13}
\mbox{\hyperlink{namespace_ink}{Ink}} files must be compiled to JSON before they can be used in-\/game. {\bfseries{This package compiles all edited ink files automatically.}} By default, compiled files are created next to their ink file.\hypertarget{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md14}{}\doxysubsubsection{Editor Compilation}\label{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md14}
This package provides tools to automate this process when a .ink file is edited.

{\bfseries{Disabling auto-\/compilation}}\+: You might want to have manual control over ink compilation. If this is the case, you can disable \char`\"{}\+Compile ink automatically\char`\"{} in the Ink\+Settings file or delete the Ink\+Post\+Processor class.

{\bfseries{Manual compilation}}\+: If you have disabled auto-\/compilation, you can manually compile all ink files using the {\bfseries{Assets \texorpdfstring{$>$}{>} Recompile \mbox{\hyperlink{namespace_ink}{Ink}}}} menu item, individually via the inspector of an ink file, or via code using Ink\+Compiler.\+Compile\+Ink().

{\bfseries{Play mode delay}}\+: By default, ink does not compile while in play mode. This can be disabled in the Ink\+Settings file.\hypertarget{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md15}{}\doxysubsubsection{In-\/game Compilation}\label{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md15}
The compiler is included in builds (See \href{\#WebGLBestPractices}{\texttt{ Web\+GL best practices}} for information on removing it), enabling you to allow the editing of ink files as part of your game.\hypertarget{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md16}{}\doxysubsection{$<$a name=\char`\"{}\+Ink\+Player\+Window\char`\"{}$>$$<$/a$>$\+Ink Player Window}\label{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md16}
The \mbox{\hyperlink{namespace_ink}{Ink}} Player Window ({\bfseries{Window \texorpdfstring{$>$}{>} \mbox{\hyperlink{namespace_ink}{Ink}} Player}}) allows you to play stories in an editor window, and provides functionality to edit variables on the fly, test functions, profile performance, save and load states, and divert.

To play a story, click the \char`\"{}play\char`\"{} button shown on the inspector of a compiled ink file, or drag a compiled ink story Text\+Asset into the window.

{\bfseries{Editor Attaching}}\+: Attaching the Ink\+Story instance used by your game to the \mbox{\hyperlink{namespace_ink}{Ink}} Player window allows you to view and edit your story as it runs in game.

See Basic\+Ink\+Example\+Editor.\+cs in the Examples folder for an example of how to\+:
\begin{DoxyItemize}
\item Show an attach/detach button on an inspector
\item Automatically attach on entering play mode
\end{DoxyItemize}

When you edit ink files, the compiler will also update the corresponding compiled .json file. If no compiled file existed before, Unity will also create a meta file for it. It is recommended that you always commit both ink and json files at the same time to avoid the file being re-\/compiled by your team members.

Adding or removing ink files will also make changes to the Ink\+Library file, and we could recommend authors also commit this file for the same reasons.\hypertarget{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md22}{}\doxysection{$<$a name=\char`\"{}\+Web\+GLBest\+Practices\char`\"{}$>$$<$/a$>$\+Web\+GL best practices}\label{md__d___repos__vertical_slice__assets__ink__r_e_a_d_m_e_autotoc_md22}
Web\+GL builds should be as small as possible. The ink compiler is included in builds, but is typically only used in the editor.
